# JASMINE warpfield demonstration code
An experimental code to simulate the image warp function.


## Installation
Clone this.


### Dependencies

- numpy
- scipy
- astropy
- astroquery


## Usage

- Input
    - list of astronomical sources
    - optics parameters
        - pointing
        - position angle
        - focal length
        - distortion function
    - detector parameters
        - number of detectors
        - offsets
        - dimension
        - pixel displacement
- Output
    - object positions on the focal plane.
